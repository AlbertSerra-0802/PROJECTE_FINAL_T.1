# PROJECTE_FINAL_T.1
Aquest projecte consisteix en el desenvolupament d’un lloc web complet inspirat en un disseny creat amb Figma. El web inclou una pàgina principal (Home), una pàgina de Contacte i diverses seccions creades amb HTML5, CSS3 i Bootstrap 5.

L’objectiu del projecte és replicar fidelment el disseny, utilitzant components responsius i bones pràctiques de maquetació.

# Tecnologies utilitzades
  **HTML5**
  **CSS**
  **Bootstrap** 

  — Cards
  — Navbars
  — Buttons
  — Utilitats (padding, margin, colors…)
